# ECG_Arrhythmia_FPGA_Accelerator
AI Approach for ECG Arrhythmia Detection with FPGA Accelerator using Vitis HLS, Vivado and PYNQ-Z2

This project implements an FPGA accelerated neural network for ECG arrhythmia classification.

## Technologies Used
- Python
- Vitis HLS
- Vivado
- PYNQ-Z2 FPGA

## Workflow
1. ECG preprocessing and feature extraction
2. Neural network training
3. Weight conversion to C header file
4. Vitis HLS hardware accelerator design
5. Vivado IP integration with AXI interface
6. Deployment on PYNQ-Z2 board

## Hardware Results
Real-time ECG classification executed on FPGA accelerator with outputs visualized through PYNQ Jupyter interface.
